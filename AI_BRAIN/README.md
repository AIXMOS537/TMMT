# AI Brain Operating System

Central command system for business operations, memory, automation, and decision support.

## Core Directories

- `OPERATIONS`: Daily priorities, active tasks, schedules, follow-ups, and command center notes.
- `FLEET`: Vehicle records, inspections, maintenance, utilization, incidents, and rental logistics.
- `FINANCE`: Revenue, expenses, payments, margins, invoices, and financial reports.
- `CUSTOMERS`: CRM records, leads, active customers, communication logs, and follow-ups.
- `REPORTS`: Weekly, monthly, fleet, financial, and operational reporting.
- `AUTOMATIONS`: Local scripts, n8n workflows, cron jobs, API notes, and automation documentation.
- `SOPS`: Standard operating procedures, checklists, training notes, and process documentation.
- `LEGAL`: Agreements, compliance documents, insurance, contracts, and risk records.
- `DATA`: Source data, exports, databases, spreadsheets, and structured datasets.
- `ARCHIVE`: Inactive records, completed projects, old reports, and historical files.

## Operating Rule

Every important business item should have one clear home, one owner, one status, and one next action.
