# TMMT Management Execution Hub

This folder is the daily control room for all three businesses.

Open these files first:

1. `TODAY.md` - what to do today.
2. `WEEKLY_PLAN.md` - what matters this week.
3. `AI_PROMPTS.md` - prompts to use when asking AI for help.
4. `BUSINESS_TASKS.md` - the master task list by business.

Simple rule: do not start the day in random messages, websites, or ideas. Start in `TODAY.md`.
